## Overview

Hatch is a minimal freelancer template for developers, designers, consultants, and solo service providers who need a clean portfolio and services website without starting from a blank project.

Built with shadcn/ui and Tailwind, Hatch is available in both Next.js and Astro editions. It combines polished page layouts with reusable block sections, so you can customize the site quickly while keeping the codebase familiar and easy to extend.

## Features

- **Freelancer-focused design**: Clean portfolio and services pages for solo professionals and small studios
- **Next.js and Astro editions**: Choose the framework that fits your project
- **100+ components**: Reusable sections and UI pieces built on shadcn/ui
- **10+ pages**: Complete starter structure for a professional portfolio site
- **MDX and JSON data patterns**: Content and data are easy to update
- **Responsive layout**: Designed for mobile, tablet, and desktop views
- **SEO metadata**: Built-in metadata patterns for a production launch
- **Shadcnblocks library support**: Extend the template with additional blocks from the Shadcnblocks library
