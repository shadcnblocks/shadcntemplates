## Overview

ChatDeck is a free, open-source SaaS landing page template built for founders and developers who want to move fast without sacrificing quality. Fork it, update the copy, deploy to Vercel — done in under an hour.

## Features

- **9 Pre-built Sections** — Hero, features, video, pricing, testimonials, FAQ, team, nav, and social proof bar.
- **Shadcn UI + Radix** — Accessible components out of the box, no a11y patches needed later.
- **TypeScript Throughout** — Fully typed, no `any` shortcuts.
- **Motion Animations** — Smooth, performant animations with no CSS keyframes to manage.
- **Pricing Toggle** — Monthly/yearly switch with animated plan cards.
- **One-Click Vercel Deploy** — Fork, connect, deploy.
- **MIT License** — Free for personal and commercial use. No attribution required.
